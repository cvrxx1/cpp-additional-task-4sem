# Windows Search Index Database Analyzer

## Описание

Программа для анализа баз данных службы индексации Windows 11. Позволяет извлекать информацию о файлах, анализировать временные метки, находить дубликаты и получать статистику по типам файлов.

## Возможности

- Подключение к базам Windows.db и Windows-gather.db
- Извлечение информации о файлах (путь, тип, размер, даты)
- Анализ временной шкалы активности файлов
- Поиск дубликатов файлов
- Статистика по типам файлов
- Вывод результатов в консоль и файл
- Программа автоматически ищет базы данных по пути: C:\ProgramData\Microsoft\Search\Data\Applications\Windows\

## Структура баз данных

### Windows.db
Содержит метаданные индексированных файлов.

Основные таблицы:
- SystemIndex_*.files - информация о файлах (путь, размер, даты)
- SystemIndex_*.props - свойства файлов (автор, название, теги)

### Windows-gather.db
Содержит общую информацию об индексированных файлах и папках.

Основные таблицы:
- Gather_*.files - информация о файлах и папках
- Gather_*.directories - информация о директориях

## Технологии

- C++17
- SQLite3
- CMake
- STL

## Безопасность

- Программа работает в режиме только чтения
- Рекомендуется работать с копиями системных баз данных
- Требуются права администратора для доступа к системным файлам
